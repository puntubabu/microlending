# ArrayCombine
combine two sorted integer arrays, using react, redux, mocha, webpack

To run, 

1. ```git clone https://github.com/puntubabu/ArrayCombine.git```

2. ```npm install```

3. ```npm run dev```

Open ```localhost:8080```

For tests,

```npm run test```

Combiner will give valid outputs only if both inputs are sorted integer arrays
